About the App

Gozizigo is a restaurant discovery and navigation app that helps users find nearby places, explore details, and navigate easily using Google Maps. The app is built with modern Android tools such as Kotlin, Jetpack Compose, and Google APIs.

Developer Information

This application is developed by an independent Android developer. More features and improvements will be added over time to enhance user experience and functionality.
